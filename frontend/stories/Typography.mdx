# Typography

**Font**: Open Sans <br />
**Fallback font**: Arial <br />
**Font-size unit**: rem (1rem = 10px) <br />

_Why rem?_ <br />
Most browsers have the option to adjust your preferred font size. This feature operates independently from the page zoom feature. When the font-size is set to a relative CSS unit, all fonts are proportionately scaled up (or down) based on the font size chosen by the user. This is not the case when pixels are used to define a font-size.

Om de opzet van FE stacks zoveel mogelijk te optimaliseren werkte ik voor Sentia aan een boilerplate gebaseerd op Next.JS en React. 

Met Next.js kan je React applicaties op de server renderen, zodat er daadwerkelijke HTML pagina’s met bijbehorende SEO optimalisaties gegenereerd worden. Dit maakt de integratie met mogelijke backend systemen ook makkelijker. Ook integreerde ik de laatste versie van SASS (Dart) als SCSS modules binnen de React componenten. Dit faciliteert een grotere focus op het bouwen van de componenten, zonder al teveel zorgen over naamgeving binnen CSS. 

### Technology

- ES6 Vanilla JS
- React
- NextJS
- HTML5
- Dart Sass/CSS3
